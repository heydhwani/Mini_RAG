Retrieval Augmented Generation (RAG) is a technique that combines information retrieval with text generation. Instead of relying only on the knowledge stored inside a language model, RAG retrieves relevant information from external documents and uses it to generate more accurate answers.

A typical RAG pipeline consists of several steps. First, documents are split into smaller chunks. Then embeddings are created for those chunks using an embedding model. These embeddings are stored in a vector database such as FAISS.

When a user asks a question, the system converts the query into an embedding and searches the vector database to find the most relevant chunks. These chunks are then provided as context to a language model, which generates the final answer.

RAG systems are widely used in building AI assistants, knowledge base chatbots, and document question-answering systems.
